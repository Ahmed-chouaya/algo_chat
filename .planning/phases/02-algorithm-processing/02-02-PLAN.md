  <action>
Create src/processing/step_extractor.py:

1. Define Pydantic models:
```python
class Variable(BaseModel):
    name: str
    type: str  # "int", "float", "list", "matrix", etc.
    initial_value: Optional[str]
    description: Optional[str]

class AlgorithmStep(BaseModel):
    step_number: int
    description: str  # Plain language explanation
    code_equivalent: str  # Pseudocode for this step
    variables: list[Variable]
    control_flow: Optional[str]  # None, "for_loop", "while_loop", "if", "elif"
    confidence: str  # "high", "medium", "low"
    confidence_reason: Optional[str]  # Why this confidence level
```

2. `extract_steps(processed_input: ProcessedInput, provider: str) -> list[AlgorithmStep]`:
   - Send processed text + extracted LaTeX to LLM
   - Use structured output (JSON mode) for consistent parsing
   - Request confidence scoring with reasoning
   - Parse response into AlgorithmStep list

3. Prompt should include:
   - "Extract the algorithm as numbered steps"
   - "For each step, provide plain language explanation"
   - "Identify variables with types and initial values"
   - "Mark control flow: loops, conditionals"
   - "Rate confidence: high (unambiguous), medium (some interpretation), low (multiple valid interpretations)"
   - "Explain confidence_reason for medium/low"

Call LLM using the provider pattern established in Phase 1.
