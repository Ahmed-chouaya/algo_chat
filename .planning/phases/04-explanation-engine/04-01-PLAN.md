Add `generateExplanation` function that calls LLM to generate explanations. The function should:

```typescript
export interface ExplanationState {
  summary: string;  // Plain language algorithm summary
  stepExplanations: StepExplanation[];  // Each step explained
  codeExplanation: string;  // What the generated code does
  generatedAt: Date;
}

export interface StepExplanation {
  stepNumber: number;
  explanation: string;
}

export async function generateExplanation(
  steps: AlgorithmStep[],
  generatedCode: string | null
): Promise<ExplanationState>
```

Use the existing LLM provider from settingsStore (same pattern as Phase 2's step extraction). The prompt should request:
1. One-paragraph summary of what the algorithm does
2. Plain language explanation for each major step
3. Explanation of what the generated Python code does (not how)
