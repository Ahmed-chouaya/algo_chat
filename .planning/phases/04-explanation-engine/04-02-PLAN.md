```typescript
export interface ChatContext {
  algorithmSummary: string;
  steps: AlgorithmStep[];
  codeExplanation: string;
  generatedCode: string | null;
}

export async function chatAboutExplanation(
  question: string,
  context: ChatContext,
  history: ChatMessage[]
): Promise<string>
```

The function should:
1. Build a prompt that includes:
   - The algorithm summary
   - Step details
   - Code explanation
   - Previous chat history
   - The user's follow-up question
2. Call the LLM with this context
3. Return the assistant's response

This enables contextual answers like "Why does step 3 use recursion?" by including step 3's explanation in context.
